# 学习通复习显示隐藏答案 HideShowChaoXingAnwer
## 作用
隐藏/显示作业详情页的答案，用于自己期末做题测试复习效果。思考之后再看正确答案。
<div align="center">
    <img src="pics/ShowEffect.gif" alt="效果演示" width="300">
</div>

## 使用方法
通过单击键盘左上方的 <kbd>~</kbd> 键来隐藏/显示正确答案，用于期末复习
